BlackboxJS is a fork of ngScenario that tries to achieve full end to end testing in any MVC based Javascript framework.

The project is available on Github

Motivation

Main motivation : a functionnal test tool of a single page application should correctly check both input (visual content) and output (REST calls) of the system under test.

Functionnal testing should be easy to read and write, so that businness and devs can talk about it.

Such a tool must be technology agnostic, so that the functionnal test suite could be kept "as is" when upgrading or changing MVC framework.

Usage

ngScenario

BlackboxJS is a fork of ngScenario, so all docs about how to install, how to run, etc, applies.

If you are very new with e2e UI testing, you might want to read this

Added inputs

To act on the system under test, you should use the jQuery passthrough function : 

jQueryFunction(selector, jQueryFunctionName, arg1, arg2, ...);
jQueryFunction('.picker-col-id-id', 'focus');

To use the enter key, use the fireEnterOn

fireEnterOn('input#new-todo');

Other keyboard input or passthrough will be added in future release.

Added outputs

A matcher was added to check last requests : 

expect(lastRequest("POST").body()).toEqual({"title": "anotherTodo","completed": false,"id": 4});

which is also available for PUT and DELETE. GET is not available, because there is no much need here to check GET request : every GET is in theory made to be displayed, thus assertion can be made in HTML.

A matcher was added for localStorage, available under the name windowLocalStorage

expect(windowLocalStorage("todos-" + QueryString.fw).getItem()).not().toContain("editing");

Examples

Run the samples

  1. Download or clone the project
  2. Install node and npm
  3. Go to the example folder and run npm install
  4. Run node app.js
  5. Open browser at http://localhost:3000 and follow the instructions

Classic TodoMVC

The great classic TodoMVC project, left "as is", only one test suite written to test all frameworks. The code for this example is available here

Available implementation example with : AgilityJS, AtmaJS, BackboneJS, EmberJS, CanJS, Dermis, Dijon, KnockoutJS, jQuery, Thorax.

RESTful TodoMVC

The TodoMVC project, with a RESTful adapter, so that request are fetch/sent to the server in a RESTful manner. The code for this example is available here.

Available implementation example with : KnockoutJS, jQuery.

Limitations

The system under test runs in a iframe, which could be sometime a problem.

Mouse gesture can not be simulated.

As of now, only MVC frameworks that integrates nicely with jQuery can use the tool.

The tool was tested under Chrome only.
