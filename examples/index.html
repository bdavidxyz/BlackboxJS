BlackboxJS examples

BlackboxJS is a fork of ngScenario that tries to achieve full end to end testing in ANY MVC based Javascript framework.

The project is available on Github

introduction

Most of the UI testing framework are exercising visual components (button, textarea...) as an input, and asserting visual component as an output. While correct, it is often not enough.

In the case of Single Page Application, the responsibility of the UI is also to send the correct data to the server

Both examples below are asserting that data are correctly sent to the server.
